Manual host
Applying rewrites the plugin config and restarts the plugin. Requires Signal K admin login.
Autopilot Provider (one-socket mode)
When ON, this plugin registers itself as the SK Autopilot Provider (WilhelmSK / freeboard / KIP connect to it via /signalk/v2/api/vessels/self/autopilots). You must also disable the 'pypilot-autopilot-provider' plugin in SK Admin - two providers would fight for the pilot ID. Benefit: one socket less to the Pi Zero TinyPilot.
Emergency: pause client
If pypilot_web on the TinyPilot chokes (Pi Zero maxed out, other clients slowing to a crawl), pausing here disconnects our socket without disabling the plugin. Resume when things settle.
Nudge steps
Degrees added to the target each time you press a nudge button. Also configurable in Signal K Admin -> Plugin Config -> PyPilot New-UI.
